# Portal Intellium Frontend 🚀

## Description 📄
Portal Intellium Frontend is a private web application designed to provide an intuitive and user-friendly interface for managing various aspects of the Intellium platform. It includes features such as call request management, ticketing, and a scrum board.

## Features ✨ 
- 📞 Call Request Management
- 📝 Ticketing System
- 📊 Scrum Board
- 🔍 Advanced Search
- 📅 Calendar Integration
- 👥 User Profile Management
- 📈 Project Management
- 🛠️ Parameter Management
- 👥 Team Management
- 🗂️ Asset Management
- ✅ Approval Processes
- 📅 Holiday Information
- ⏱️ Time Tracking

## Installation 💻
To install and run the project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/portal-intellium-frontend.git
    ```
2. Navigate to the project directory:
    ```bash
    cd portal-intellium-frontend
    ```
3. Install the dependencies:
    ```bash
    npm install
    ```

## Usage 🚀
To start the development server, run:
```bash
npm start
```
This will launch the application in your default web browser. You can now start developing and testing your application.
